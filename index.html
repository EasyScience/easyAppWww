Build graphical applications for data analysis and modelling easily

Application skeleton with eye-catching and intuitive graphical interface for the easyScience framework. Quickly build all-in-one cross-platform applications using the provided collection of Qt QML components, Python tools, and by binding with the easyCore Python backend.

Current success stories are for the fields of powder neutron diffraction (easyDiffraction) and reflectometry (easyReflectometry), where user defined models are used for simulations and can be refined against experimental data.

Features

Free and open-source

Anyone is free to use easyApp and the source code is openly shared on GitHub.

Cross-platform

easyApp-based applications works across operating systems: macOS, Windows, or Linux.

Easy to use

Intuitive tabbed interface with a clear workflow, built-in step-by-step user guides and video tutorials help speed up data modelling and analysis.

Everything is included

easyApp-based applications is distributed as an all-in-one package that includes all the dependencies and can be installed with only a few clicks.
